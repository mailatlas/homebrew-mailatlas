# MailAtlas Homebrew Tap

Homebrew tap for the MailAtlas CLI.

- Package repository: https://github.com/mailatlas/mailatlas
- Documentation: https://mailatlas.dev/docs
- Support policy: https://github.com/mailatlas/mailatlas/blob/main/SUPPORT.md
- Security policy: https://github.com/mailatlas/mailatlas/security/policy

## Install

```bash
brew tap mailatlas/mailatlas
brew install mailatlas
mailatlas doctor
```

If Homebrew resolves a different formula named `mailatlas`, use the fully qualified formula name:

```bash
brew install mailatlas/mailatlas/mailatlas
```

## Formula

The formula lives at `Formula/mailatlas.rb`.

The core `mailatlas/mailatlas` release workflow updates this tap after publishing a tagged release.
